Quovie: This app will display a list of movies with an image and a title and when the index of that movie within the listView is clicked it will then open another page that will display an image of the movie and a description along with a rating. The user will then be able to select from the list of movies which ones he or she would like to save to their personal collection. And the user will be able to view this collection in another page of the app.
